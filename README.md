## Описание программы

**Movie Library** – это графическое приложение на Python (Tkinter) для ведения личной коллекции фильмов. Программа позволяет:
- добавлять фильмы (название, жанр, год выпуска, рейтинг);
- просматривать все добавленные фильмы в таблице;
- фильтровать фильмы по жанру и году выпуска;
- удалять выбранный фильм;
- автоматически сохранять все данные в файл `movies.json` и загружать их при запуске.

Интерфейс интуитивно понятен, данные валидируются (год – целое число от 1800, рейтинг – число от 0 до 10).

## Инструкция по использованию

### Добавление фильма
1. Заполните поля:
   - **Название** (обязательно)
   - **Жанр** (необязательно, по умолчанию `"Не указан"`)
   - **Год** (целое число, ≥ 1800)
   - **Рейтинг** (число от 0 до 10)
2. Нажмите кнопку **«Добавить фильм»** – запись появится в таблице.
3. Если поле «Жанр» пустое, автоматически установится значение `"Не указан"`.

### Фильтрация
- Введите жанр (можно частично) и/или год в соответствующие поля фильтра.
- Нажмите **«Применить фильтр»** – таблица обновится, показывая только подходящие записи.
- Нажмите **«Сбросить»** – отобразятся все фильмы.

### Удаление фильма
1. Выделите строку в таблице (кликните по ней).
2. Нажмите кнопку **«Удалить выбранный»** – фильм исчезнет из таблицы и из файла.

## Примеры работы

### Добавление фильма
- **Название:** "Матрица"
- **Жанр:** "Фантастика"
- **Год:** 1999
- **Рейтинг:** 8.7

Нажать **«Добавить фильм»** → строка появляется в таблице.

### Фильтрация по жанру
В поле «Жанр» ввести `"Фантастика"` → нажать **«Применить фильтр»** → останутся только фильмы с таким жанром.

### Удаление
Выделить фильм, нажать **«Удалить выбранный»** – запись удаляется.
